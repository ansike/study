# 百万级数据排序

# 生成数据

生成 10 个随机 100w 级的数据 json 文件,包含 id 和 age(0-99)

```shell
node generateData.js
```

# 创建桶, 并顺序读取文件, 存入桶中, 计算每个桶中的数据情况

node bucketSort10.js
